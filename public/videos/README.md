# ویدیوی دمو

صفحه‌ی `/demo` دنبال این فایل می‌گردد:

```
public/videos/demo.mp4
```

یک اسکرین‌رکورد کوتاه (۶۰ تا ۹۰ ثانیه) از پنل بگیرید که این مسیر را نشان دهد:
داشبورد ← واحدها ← صدور قبض ← پرداخت ← اطلاعیه‌ها ← گزارش‌ها.

پیشنهاد فنی:
- فرمت `mp4` با کدک `H.264` و صدای `AAC` (بیشترین سازگاری با مرورگرها)
- رزولوشن ۱۹۲۰×۱۰۸۰، نرخ بیت حدود ۲.۵ مگابیت بر ثانیه
- برای حجم کمتر می‌توانید نسخه‌ی `webm` هم کنار آن بگذارید: `demo.webm`

تصویر پوستر (قبل از پخش) از `public/images/hero-building-night.webp` خوانده
می‌شود؛ اگر پوستر اختصاصی دارید، `public/videos/demo-poster.webp` را بسازید و
مسیر را در `resources/js/pages/demo/DemoPage.tsx` عوض کنید.

تا وقتی این فایل اضافه نشده، صفحه به‌جای پخش‌کننده یک پیام راهنما نشان می‌دهد
و بقیه‌ی صفحه (توضیح امکانات) عادی کار می‌کند.
